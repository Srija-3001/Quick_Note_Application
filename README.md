# Quick Note Application

## Description

A simple full-stack web application where users can write and save notes. Notes are stored in an SQLite database and displayed on the same page.

## Features

* Add notes
* Store notes in SQLite database
* Display saved notes
* Simple and responsive UI
* Flask backend

## Technologies Used

* Python
* Flask
* SQLite
* HTML
* CSS

## How to Run

1. Install Flask:
   pip install flask

2. Run the application:
   python app.py

3. Open:
   http://127.0.0.1:5000

## Project Structure

* app.py
* templates/index.html
* static/style.css
* notes.db
